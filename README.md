# multimodal_playground

A small **`playground`** repository for quick prototyping of **multimodal training / evaluation** ideas.
The code is intentionally lightweight: a `main.py` entrypoint, a `run_main.sh` runner, plus `src/` and `scripts/`.

## Repository structure

- **`src/`** — core code (models, data, losses, helpers).
- **`scripts/`** — run helpers / utilities.
- **`main.py`** — main entrypoint.
- **`run_main.sh`** — example runner script.
- **`extract_embs.ipynb`** — notebook for extracting / inspecting embeddings.
- **`README.md`** — this file.
